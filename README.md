# 📝 To-Do List App

A simple and modern To-Do List web app built using HTML, CSS, and JavaScript.

## 🚀 Features
- Add tasks
- Delete tasks
- Mark tasks as completed
- Responsive design

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 📂 Project Structure
```
├── index.html
├── style.css
└── script.js
```
